Легендарные туры AC/DC

Группа AC/DC известна своими энергичными живыми выступлениями. Вот несколько их самых значимых туров:

1. Highway to Hell Tour (1979-1980)

Последний тур с Боном Скоттом, который сделал AC/DC знаменитыми по всему миру.

2. Back in Black Tour (1980-1981)

Первый тур с Брайаном Джонсоном, который стал триумфальным возвращением группы.

3. The Razors Edge Tour (1990-1991)

Грандиозный тур в поддержку культового альбома The Razor's Edge.

4. Black Ice World Tour (2008-2010)

Один из самых успешных туров AC/DC, собравший миллионы фанатов по всему миру.

5. Rock or Bust World Tour (2015-2016)

Последний большой тур, который доказал, что AC/DC остаются одной из лучших рок-групп в истории.
